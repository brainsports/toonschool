# 툰스쿨 키워드 생성 규칙

툰스쿨 에디터의 주제 만들기 단계에서 학생이 선택할 이야기 키워드를 생성할 때 지켜야 할 규칙입니다.
이 규칙은 AI 모델 프롬프트와 클라이언트 후처리 로직 모두에 적용됩니다.

---

## 1. 기본 원칙

키워드는 단순히 **문법상 명사**이면 되는 것이 아닙니다.
키워드는 **초등학생이 만화 이야기 소재로 바로 사용할 수 있는 쉬운 명사 또는 명사구**여야 합니다.

| 기준 | 설명 |
|------|------|
| 장면화 가능 | 학생이 만화 장면으로 바로 떠올릴 수 있어야 함 |
| 이야기 소재화 | 만화 속 인물, 장소, 사건, 물건, 행동 주제로 바꾸기 쉬운 말 |
| 쉬운 말 | 초등학생이 바로 이해하는 단어 |
| 길이 | 2~8글자 중심, 필요하면 2어절 명사구 허용 |

---

## 2. 좋은 키워드 예시

```
우리나라, 국토, 지도, 산, 강, 바다, 섬, 도시, 마을, 여행,
길찾기, 고장, 자연, 생활, 친구, 대화, 배려, 경청, 약속, 갈등 해결,
산맥, 평야, 해안, 갯벌, 항구, 농촌, 어촌, 계절, 날씨, 기후
```

## 3. 절대 금지 유형

### 3.1. 형용사 (반드시 삭제)
- 다양한, 새로운, 중요한, 알맞은, 바른, 올바른, 여러, 다른, 같은

### 3.2. 동사 어간 (어미가 잘린 형태 — 반드시 삭제)
- 살펴보, 알아보, 생각하, 대화하, 비교하, 설명하, 찾아보, 이해하, 공부하

### 3.3. 어미가 붙은 동사형 표현 (반드시 삭제)
- 살펴보기, 알아보기, 비교하기, 설명하기, 대화하고, 배려하며, 알아보며, 공부하기

### 3.4. 조사로 끝나는 표현 (반드시 삭제)
- 바탕으로, 중심으로, 때문에, 위해, 에서, 에게, 보다, 으로, 와/과 함께

### 3.5. 추상적이거나 분석용 단어 (그대로 쓰지 않고 변환)
- 분포, 특징, 관계, 영향, 변화, 요소, 자료, 활동, 내용, 방법

## 4. 실패 사례와 올바른 변환 예시

| 잘못된 키워드 | 이유 | 올바른 변환 |
|---|---|---|
| `다양한` | 형용사 | 삭제 |
| `살펴보` | 동사 어간 | `탐험` 또는 `관찰` |
| `분포` | 너무 추상적 | `지도` 또는 `지역` |
| `자료` | 너무 추상적 | `사진` 또는 `지도` |
| `활동` | 너무 추상적 | `여행` 또는 `체험` |
| `방법` | 단독 사용 금지 | `대화 방법`처럼 구체 명사구로만 허용 |
| `특징` | 단독 사용 금지 | `산과 강`, `도시와 농촌`처럼 구체어로 변환 |
| `변화` | 단독 사용 금지 | `계절 변화`처럼 구체 명사구로만 허용 |
| `내용` | 너무 포괄적 | 삭제 |
| `알아보기` | 어미 붙은 동사형 | 삭제 |

## 5. AI 프롬프트 필수 문장

AI를 통해 키워드를 추천받을 때, 프롬프트의 [중요 조건]에 아래 문장을 반드시 포함해야 합니다.

> "키워드는 초등학생이 만화 이야기 소재로 바로 사용할 수 있는 쉬운 명사 또는 명사구로만 작성한다. 형용사, 동사 어간, 어미가 붙은 말, 조사로 끝나는 말, 추상적인 분석어는 금지한다. '다양한, 살펴보, 분포, 특징, 관계, 영향, 변화, 요소, 자료, 활동, 내용' 같은 단어는 그대로 쓰지 않는다. 필요하면 지도, 여행, 산, 강, 도시, 마을, 친구, 대화처럼 장면으로 그릴 수 있는 구체적인 말로 바꾼다."

## 6. 후처리 필터링 기준 (filterAndCleanKeyword)

### 6.1. 절대 금지어 목록 (완전 일치 시 null 반환)
형용사, 동사 어간, 추상어를 포함한 아래 목록에 해당하면 즉시 제거합니다.

**형용사형:** 다양한, 새로운, 중요한, 알맞은, 바른, 올바른, 여러, 다른, 같은

**동사 어간 패턴 (어미 없이 끝남):** 살펴보, 알아보, 생각하, 대화하, 비교하, 설명하, 찾아보, 이해하, 공부하

**추상 분석어:** 분포, 영향, 요소, 내용

**포괄적 학습 용어:** 학습, 단원, 주제, 수업, 평가, 목표, 성취, 기준, 이해

### 6.2. 어간 변환 (특정 어간 → 구체 명사)
잘못 잘려진 어간은 문맥에 맞는 자연스러운 명사로 치환합니다.

| 패턴 | 변환 |
|---|---|
| `알아보`로 끝남 | → `이해` |
| `바탕으`로 끝남 | → `바탕` |
| `살펴보`로 끝남 | → `탐험` |

### 6.3. 어미 및 조사 제거
**동사형/어미:** 하며, 하고, 하는, 되며, 되고, 하다, 하
**조사/의존명사:** 으로, 에서, 에게, 보다, 처럼, 위해, 대한, 을, 를, 이, 가, 은, 는, 와, 과, 의, 에, 로, 부터, 까지, 과함께, 와함께

---

## 7. 안전 기본 키워드 (Fallback)

AI 생성에 실패하거나 유효한 키워드가 부족한 경우, 과목/단원명 기반의 안전한 기본 키워드로 보충합니다.

- 사회 기본: 지형, 국토, 환경, 기후, 산업, 교통, 인구, 도시, 농촌, 바다
- 국어 기본: 인물, 마음, 대화, 장면, 표현, 상상, 친구, 사건, 감정, 이야기
- 과학 기본: 실험, 관찰, 생물, 환경, 에너지, 로봇, 우주, 발명, 동물, 식물

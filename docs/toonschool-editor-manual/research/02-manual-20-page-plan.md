## 2. 기존 계획에서 수정한 내용과 이유

1. **3쪽 “시작하기”**: 학생 로그인 후 첫 화면이 `/student/mypage` 임을 반영. 마이페이지의 `툰스쿨 에디터 입장` 버튼으로 에디터 진입. (기존 안의 “학생 첫 화면”이 대시보드로 가정한 부분 정정.)
2. **6쪽 “핵심어 선택”을 단원선택이 아닌 주제 만들기 아래로 배치**: 실제로 핵심어(키워드) 선택은 `/student/topic` 페이지의 서브 단계. 단원 선택에는 핵심어 없음.
3. **16쪽 “세상 속 이야기 만들기”**(기존 “생활 속 이야기 만들기”): 실제 화면 명칭이 “세상 속 이야기”이며 `생활 연결`은 그 안의 탭. 화면 명칭에 맞춰 제목 수정.
4. **16·17쪽이 같은 `/student/unit-summary` 페이지의 두 섹션**: 세상 속 이야기(탭 3종)와 OX 문제 5개가 한 페이지에 있으나, 내용이 많아 2쪽으로 분리(16: 세상 속 이야기, 17: OX 퀴즈).
5. **18쪽 “뒤표지와 작품 완성”**: 뒤표지 꾸미기 → 만화보기 진입(완료 보상)까지 한 흐름으로 묶음.
6. **19쪽 “작품 보기·공유·문제 해결”**: 만화보기 플립북 + 공유 링크 생성 + 마이페이지 작품 목록 + 정상 안내 화면을 한 쪽에 정리.
7. 단원 정리의 OX는 “만들기”(학생이 5문제 생성·수정)이므로 “OX 퀴즈 만들기”로 명칭 유지(실제 헤딩 “OX 문제 5개 만들기”).

## 3. 페이지별 핵심 목적 / 왼쪽 이미지 / 오른쪽 설명 (요약)

- **2쪽 소개**: 5개 산출물(표지·6컷 만화·세상 속 이야기·OX 퀴즈·뒤표지) 흐름 소개. 왼쪽=썸네일 모음, 오른쪽=특징 설명.
- **3쪽 시작하기**: 로그인→마이페이지→에디터 입장. 왼쪽=로그인/마이페이지 캡처, 오른쪽=진입 순서. (비밀번호 영역 캡처 금지.)
- **4쪽 전체 과정**: 8단계 흐름도. 왼쪽=사이드바 캡처 또는 흐름도, 오른쪽=단계 요약.
- **5쪽**: 1·2단계 선택. 왼쪽=선택 전/후 캡처 2장, 오른쪽=번호별 순서.
- **6쪽 핵심어**: 키워드 추천/선택. 왼쪽=추천 목록+선택 상태, 오른쪽=선택 방법(최대 4개).
- **7쪽 주제**: 주제 생성/선택. 왼쪽=생성 전/결과, 오른쪽=선택+다음.
- **8쪽 대본 생성**: AI 생성. 왼쪽=생성 전/중/결과, 오른쪽=생성 버튼 안내.
- **9쪽 대본 수정**: 컷 편집/핵심개념/표지대화. 왼쪽=편집 화면, 오른쪽=수정 도구 설명.
- **10쪽 표지**: 캔버스 에디터. 왼쪽=편집 화면, 오른쪽=완성 버튼 안내.
- **11쪽 에디터 개요**: 만화제작 전체 화면. 왼쪽=전체 캡처, 오른쪽=도구/캔버스/줌 설명.
- **12쪽 배경**: 배경 도구. 왼쪽=생성 전/중/완료, 오른쪽=`배경 모두 생성`/재생성(컷당 1회) 안내.
- **13쪽 등장인물**: 캐릭터 도구. 왼쪽=선택/배치, 오른쪽=드래그·크기 조절.
- **14쪽 말풍선/글**: 대사/말풍선 도구. 왼쪽=추가/입력, 오른쪽=위치·크기.
- **15쪽 6컷 완성**: 전체 6컷 확인. 왼쪽=6컷 완료 화면, 오른쪽=`단원 정리 →` 이동.
- **16쪽 세상 속 이야기**: 3탭(역사/최신/생활연결). 왼쪽=생성 결과, 오른쪽=탭 전환·완료.
- **17쪽 OX 퀴즈**: 5문제 생성/수정. 왼쪽=문제 목록, 오른쪽=정답 토글·수정.
- **18쪽 뒤표지/완성**: 폼+색상. 왼쪽=뒤표지 편집, 오른쪽=`만화 보기` 완료.
- **19쪽 보기/공유**: 플립북+공유+마이페이지. 왼쪽=만화보기/공유 모달, 오른쪽=링크 복사·목록.
- **20쪽 마지막**: 완성 작품 표지/펼침 화면. 버전·문의 안내.

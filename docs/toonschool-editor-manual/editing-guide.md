# 툰스쿨에디터 사용설명서 수정 안내서

> 본 설명서는 PPTX(디자인 기준) + DOCX(문장 원본) + PDF(배포본) 3종으로 구성.
> 화면이나 기능이 바뀌면 전체를 다시 만들지 말고 해당 페이지의 이미지·설명만 교체하세요.

## 0. 파일 위치
- PPTX: `source/toonschool-editor-manual-v1.0.pptx`
- DOCX: `source/toonschool-editor-manual-v1.0.docx`
- PDF: `pdf/toonschool-editor-manual-v1.0.pdf`
- 원본 캡처: `screenshots/originals/`, 선정본: `screenshots/selected/`
- 원고: `copy/manual-copy.md`

## 1. PPTX 수정 방법
- **화면 이미지 교체**: 해당 그림을 선택 → 우클릭 ‘그림 변경’ → 새 PNG 선택. 동일한 이미지 프레임(크기/위치)이 유지돼 찌그러지지 않아요.
- **이미지 비율 유지**: 그림 서식 → ‘자르기-비율’으로 원본 비율 맞춤. 캔버스에 맞춰 채우기(Fill) 권장.
- **번호·화살표 이동**: 번호 원/화살표는 별도 도형. 드래그로 이동, 서식으로 색 변경.
- **설명 문장 수정**: 텍스트 상자를 눌러 직접 수정.
- **페이지 복제**: 슬라이드 우클릭 → ‘슬라이드 복제’.
- **페이지 삭제/추가**: 슬라이드 창에서 삭제 또는 기존 슬라이드 복제 후 편집. 총 20쪽 유지.
- **새 페이지 추가**: 본문 레이아웃 슬라이드를 복제해 동일 구조(상단 STEP/제목, 왼쪽 이미지, 오른쪽 설명, 하단 페이지 번호) 유지.
- **PDF 다시 출력**: 파일 → 내보내기 → PDF 만들기. A4 가로형, 20쪽 확인.

## 2. DOCX 수정 방법
- **이미지 교체**: 셀 안의 그림 선택 → 우클릭 ‘그림 변경’. 2열 표 구조가 유지돼 위치가 안 무너져요.
- **설명 문장 수정**: 번호별 문단을 직접 수정.
- **페이지 나누기 유지**: 각 페이지 끝에 수동 페이지 나누기 삽입(단축키 Ctrl+Enter). 삭제하지 마세요.
- **2열 구조 유지**: 페이지별 2열 표(왼쪽 이미지/오른쪽 설명)를 그대로 사용.
- **글자 크기**: 본문 12pt 이하로 줄이지 마세요(가독성).
- **페이지 번호·버전**: 꼬리말(footer)에서 수정. 표지는 번호 생략 가능.

## 3. PDF 갱신 방법
1. PPTX(또는 DOCX)에서 수정.
2. 표지/마지막 페이지의 버전 번호 변경(예: v1.0 → v1.1) + `change-log.md` 항목 추가.
3. PPTX → PDF 내보내기(A4 가로형).
4. 20쪽 여부, 빈 페이지 없음, 이미지 선명도, 페이지 번호 확인.
5. 이전 PDF와 파일명 구분(`-v1.1.pdf`) 또는 덮어쓰기 후 `change-log.md`에 기록.

## 4. 버전 번호 규칙
- 화면/기능 변경 반영: v1.1, v1.2 … (마이너).
- 오탈자/문구 수정만: v1.0.1 등 패치.

## 5. 수정 절차 체크리스트
1. 변경된 기능 확인(`research/01` 대조).
2. 해당 화면 다시 캡처(`03-screenshot-checklist.md` 파일명 사용).
3. PPTX ‘그림 변경’으로 교체.
4. 번호·화살표 위치 조정.
5. DOCX 이미지·문장 교체.
6. `copy/manual-copy.md` 문장 수정.
7. 버전 변경 + `change-log.md` 기록.
8. PDF 다시 출력.

- 한 줄 설명: 만들고 싶은 단원을 골라요.
- 라우트: `/student/select-unit`

### 왼쪽 이미지
- 메인 캡처: 05-unit-selected.png (2단계 선택 완료 화면)
- 서브 캡처: 05-unit-step1.png (1단계)
- 이미지 위 번호(최대 4):
  - ① 학년 선택 칸(초3~초6) — 번호 원(주색) + 강조 테두리
  - ② 학기 선택 칸(1학기/2학기) — 번호 원
  - ③ 과목·대단원·중단원 드롭다운 — 번호 원 + 강조 테두리
  - ④ ‘주제 만들기 ✨’ 버튼 — 번호 원 + 화살표

### 오른쪽 번호별 설명
1. 학년과 학기를 골라요. (예: 초4, 1학기)
2. ‘다음 단계 🚀’를 눌러요.
3. 과목과 대단원, 중단원을 골라요. (예: 과학)
4. ‘주제 만들기 ✨’를 누르면 다음으로 넘어가요.

- 내용: 선생님이 정해 준 학년·단원만 골라야 할 수도 있어요. 그땐 다른 학년은 누를 수 없어요.

## B. 11쪽 — 만화 제작 화면 알아보기

- STEP: 11
- 페이지 제목: 만화 제작 화면 알아보기
- 한 줄 설명: 만화를 만드는 화면을 알아봐요.
- 라우트: `/student/comic/full`

### 왼쪽 이미지
- 메인 캡처: 11-comic-full.png (만화제작 전체 화면)
- 이미지 위 번호(최대 5):
  - ① 왼쪽 세로 도구바(선택·캐릭터·대사·말풍선·배경·레이어) — 번호 원 + 강조 테두리
  - ② 가운데 2×3 그리드 캔버스(6컷) — 번호 원
  - ③ 컷 제목/대사 표시 영역 — 번호 원
  - ④ 오른쪽 아래 줌 컨트롤(+/-·화면 맞춤) — 번호 원 + 화살표
  - ⑤ 하단 핵심정리(핵심 개념 3개) — 번호 원

### 오른쪽 번호별 설명
1. 왼쪽 세로 도구로 그림을 꾸며요. (선택·캐릭터·대사·말풍선·배경·레이어)
2. 가운데 2×3 칸에 6컷 만화가 만들어져요.
3. 컷을 누르면 선택되고, 두 번 누르면 크게 볼 수 있어요.
4. 오른쪽 아래 +/- 로 화면을 확대·축소하고 ‘화면 맞춤’으로 맞춰요.
5. 만든 내용은 자동으로 저장돼요.

### 주의 상자(주황)
- 제목: 주의하세요
- 내용: 만화제작은 최대 15분이 걸릴 수 있어요. 배경이 만들어지는 동안 기다려 주세요.

### 하단
- 이전 단계: 표지 만들기
- 다음 단계: 배경과 장면 만들기
- 페이지 번호: 11 / 20

- 서브 캡처: 19-viewer-start.png, 19-viewer-flip.png, 19-mypage-works.png

  - ① 만화보기 ‘책 펼치기’ 버튼 — 번호 원
  - ② 플립책 넘기기 화면 — 번호 원
  - ③ ‘친구에게 자랑하기’ 버튼 — 번호 원 + 강조 테두리
  - ④ 공유 링크 모달 ‘링크 복사하기’ — 번호 원 + 화살표
  - ⑤ 마이페이지 작품 목록 — 번호 원

### 오른쪽 번호별 설명
1. ‘책 펼치기’를 눌러 만화책을 넘겨 봐요.
2. ‘PDF 다운로드’로 저장하거나 ‘친구에게 자랑하기’로 공유해요.
3. 공유 링크가 만들어지면 ‘링크 복사하기’로 복사해요.
4. 복사한 링크를 친구에게 보내면 `/book/...` 주소에서 볼 수 있어요.
5. 마이페이지에서 내 작품 목록을 볼 수 있어요.

## 4쪽 — 전체 제작 과정
- 한 줄 설명: 만화책은 모두 8단계로 만들어져요.
- 캡처: 04-flow-sidebar.png
- 번호별 설명:
  1. 단원 선택 — 학년·과목·단원을 골라요.
  2. 주제 만들기 — 핵심어로 만화 주제를 정해요.
  3. 대본 만들기 — 6컷 이야기를 만들어요.
  4. 표지만들기 — 책 표지를 꾸며요.
  5. 만화제작 — 배경·인물·말풍선을 넣어요.
  6. 단원 정리 — 세상 속 이야기·OX 퀴즈를 만들어요.
  7. 뒤표지 꾸미기 — 지은이와 색을 정해요.
  8. 만화보기 — 완성된 책을 넘겨보고 자랑해요.
- 알아두세요: 왼쪽 단계 표시에서 완료한 단계에는 체크가 생겨요. 다음 단계는 잠겨 있어요.
- 이전: 시작하기 / 다음: 학년·학기·과목·단원 선택

## 5쪽 — 학년·학기·과목·단원 선택
- 한 줄 설명: 만들고 싶은 단원을 골라요.

- 번호별 설명:
  1. 학년과 학기를 골라요. (예: 초4, 1학기)
  2. ‘다음 단계 🚀’를 눌러요.
  3. 과목과 대단원, 중단원을 골라요. (예: 과학)
  4. ‘주제 만들기 ✨’를 누르면 다음으로 넘어가요.
- 알아두세요: 선생님이 정해 준 학년·단원만 골라야 할 수도 있어요.

## 6쪽 — 핵심어 선택
- 한 줄 설명: 만화에 넣을 핵심어를 골라요.
- 캡처: 06-keyword-list.png, 06-keyword-more.png, 06-keyword-selected.png
- 번호별 설명:
  1. ‘키워드 추천 ✨’를 누르면 핵심어가 나와요.
  2. 마음에 드는 핵심어를 눌러 골라요. (최대 4개)
  3. ‘키워드 2개 더 보기 ✨’로 더 볼 수 있어요.
  4. 다 골랐으면 ‘이 키워드로 주제 만들기 ✨’를 눌러요.
- 주의하세요: 핵심어는 최대 4개까지만 고를 수 있어요.

# Toonschool Project Rules

- **Language Constraint**: Always communicate, explain, and ask questions in Korean.
- **English Code Terms**: Keep code terms (like filenames, functions, variables, database tables, and columns) in their original English form (e.g., `ToonEditor.tsx`, `fetchUnits`, `curriculum_units`).
- **Question & Confirmation Template**: When asking for user confirmation or choices, always format the questions using the following template:

  ## 현재 상황
  [지금 어떤 상태인지 설명]

  ## 선택해야 할 내용
  [사용자가 선택해야 할 내용]

  ## 선택지 설명
  [각 선택지의 의미와 차이점 설명]

  ## 추천 선택
  [추천하는 번호 및 선택안]

  ## 추천 이유
  [추천 이유]

  ## 주의할 점
  [데이터 삭제, 테이블 변경, 기존 기능 손상 등의 리스크 경고]
